# HtmlTest 项目简介

## 项目概述
HtmlTest 是一个包含多个独立 Web 游戏、动画和交互效果的集合项目。项目使用纯 HTML、CSS 和 JavaScript 实现，部分项目使用了 Three.js、Matter.js 等第三方库。

## 项目类型
- **游戏类**: Helltaker、魔方系列、数独、五子棋、气球爆破、垃圾分类、物理方块弹射、老虎机
- **动画类**: 气球动画、星空动画、雨雪效果、Matrix数字雨、旋涡星空、爱心动画

## 技术栈
- **核心**: HTML5, CSS3, JavaScript (ES6+)
- **3D渲染**: Three.js (魔方游戏)
- **物理引擎**: Matter.js (物理方块弹射)
- **UI框架**: Tailwind CSS (数独、垃圾分类、五子棋)
- **图标**: Font Awesome
- **字体**: Crimson Text (Helltaker), Inter (其他游戏)

## 项目特点
1. **完全独立**: 每个文件都是一个完整的、可独立运行的 HTML 文件
2. **响应式设计**: 大部分游戏支持移动端和桌面端
3. **程序化音频**: Helltaker5050 使用 Web Audio API 生成程序化音效
4. **多样化**: 涵盖益智、动作、策略、休闲等多种游戏类型
