# 产品说明

## 项目目的
HtmlTest 是一个用于展示 Web 前端技术能力的项目集合，通过多个独立的游戏和动画效果展示 HTML5、CSS3 和 JavaScript 的编程技巧。

## 解决的问题
- 展示纯前端技术实现的多种交互效果
- 提供无需后端支持的完整游戏体验
- 演示 Canvas 2D 和 WebGL (Three.js) 的渲染能力
- 展示响应式设计和移动端适配

## 用户体验目标
- **即时可用**: 打开 HTML 文件即可运行，无需构建或安装
- **流畅交互**: 所有游戏和动画都经过优化，提供流畅的帧率
- **跨平台**: 支持桌面端和移动端浏览器
- **直观控制**: 支持键盘、鼠标和触控操作

## 目标用户
- Web 前端开发者
- 游戏开发爱好者
- 学习 HTML5/CSS3/JavaScript 的学生
- 对交互式网页设计感兴趣的用户
